To try out samr it's recommended to install it in a virtualenv.
samr is written in Python 3.3.

Pre-install numpy!
Pre-install scipy!

To install:
pip install -e path/to/samr

To install (for development):
pip install -e path/to/samr -r docs/setup/requirements-dev.txt


Download this: http://www.wjh.harvard.edu/~inquirer/inqtabs.txt
Page: http://www.wjh.harvard.edu/~inquirer/spreadsheet_guide.htm
